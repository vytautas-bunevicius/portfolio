Spotify Top 50 Tracks of 2020 Analysis
This project leverages the Numpy and Pandas libraries to conduct an in-depth analysis of the Spotify Top 50 Tracks of 2020 dataset. The aim is to extract insights into popular tracks, artists, genres, and various features through exploratory data analysis, addressing specific questions about the dataset.
Summary / Findings
This project delves into Spotify's top tracks dataset, utilizing Python libraries like NumPy and Pandas for efficient data analysis. The initial steps involve data cleaning, removing unnecessary columns like 'Unnamed: 0,' and identifying missing values. Exploring the dataset reveals insights into genres, artist frequencies, and track characteristics like danceability and loudness.
The project unveils intriguing patterns such as Pop being the most popular genre and explores the distribution of artists and their track appearances. Utilizing descriptive statistics, it identifies outliers, the longest and shortest tracks, and explores the diversity of genres, artists, and albums. The analysis uncovers that some genres are more prevalent, while others, like Dreampop/Hip-Hop/R&B, are relatively rare.
Further examinations include correlation analyses, highlighting strongly positively correlated features like loudness and energy. The project concludes by comparing danceability, loudness, and acousticness across genres, offering insights into the musical diversity present in each genre. Overall, this analysis provides a comprehensive exploration of Spotify's top tracks, unraveling trends and patterns in the world of music.
